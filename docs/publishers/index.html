Publishers

A publisher publishes content provided by one or more providers.

Publishers are free to publish content in any shape and form. They are supposed to create artifacts which can then be published. For example, a publisher might create a website, or a pdf file.

Here is a list of publishers packaged with cl-ownpress.

Publishers packaged with cl-ownpress

  • clown-blog
    Creates a static Html site which can be hosted on platforms like Github pages. Meant for publishing personal blogs.
